# Raghav Sharma Portfolio

Personal portfolio built with React and Vite.

## Tech

- React 19
- Vite
- ESLint

## Content

Portfolio content is centralized in [src/content/siteContent.json](src/content/siteContent.json).

Edit that file to update:

- hero links
- about copy and stack
- experience
- proof of work
- awards
- footer links

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
